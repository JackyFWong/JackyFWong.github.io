This is an extremely crude implementation of blackjack made in JavaScript for my GC 3400 class at Clemson University. Source code is located here.

You are playing against one computer opponent. There are modified rules.

  • Ace is worth 1 point
  • Face cards are 10 points
  • There is no dealer
  • You always go first
  • Only actions are to hit; no doubling, splitting, or surrendering
  • Opponent can stand at each of your hits
  • Get as close to 21 as possible

Start button can be used to restart the game.
